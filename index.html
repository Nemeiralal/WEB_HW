Summary: Latitude vs. X

The data for this project was collected by making API calls to OpenWeatherMap website, and more than 500 cities were studied to study the variation of climatic variables against latitude.

We used library citipy along with pandas and matplotlib to derive the figures.

As required, the output was put together on the front end through HTML using CSS and Bootstrap.
